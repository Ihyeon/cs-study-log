# Chapter 01 - 일차연립방정식
## 📅 2025-09-08

### 1. 일차연립방정식
#### ax = b  
- ```a ≠ 0``` x = a^(-1) * b  
- ```a = 0, b = 0``` 부정(무수히 많은 해)  
- ```a = 0, b ≠ 0``` 불능(해가 없음)  
#### n원 일차연립방정식
- 미지수가 n개인 일차방정식들을 유한개 묶어 놓은 것

### 2. 소거법
#### 방정식에 관한 3가지 기본 연산
1. 두 방정식을 교환한다.
2. 한 방정식에 0이 아닌 상수를 곱한다.
3. 한 방정식에 임의의 상수를 곱하여 다른 방정식에 더한다.
